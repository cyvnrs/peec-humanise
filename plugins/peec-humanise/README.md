# peec-humanise (plugin)

Claude Code skill that rewrites Peec AI tracked prompts into the language humans actually type into search. See the [top-level README](../../README.md) for install instructions and a full description.

The skill itself lives at [`skills/peec-humanise/SKILL.md`](./skills/peec-humanise/SKILL.md). The curses picker at [`skills/peec-humanise/scripts/picker.py`](./skills/peec-humanise/scripts/picker.py).
